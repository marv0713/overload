# deep-stock-analysis

用于单公司投研视频，例如 Unrivaled Investing 或 Sven Carlin 的个股拆解。

写作目标：
- 输出「炼金投研」系列文章。
- 聚焦单一公司或少数相关标的的商业模式、财务数据、估值逻辑、管理层、竞争格局和风险。
- 所有观点必须归因给视频、博主或 AI 提取，不能给出投资建议。
- 文章标题保持中性，优先使用「Ticker + 中文常用名」。
- transcript 没有直接提到的背景信息必须用「炼金旁注」标记。

结构偏好：
1. 免责声明
2. 引言与核心金句
3. 公司/主题简介
4. 核心数据与财务表现
5. 竞争格局
6. 管理层团队与风格
7. 核心观点与逻辑拆解
8. 长短期催化因素
9. 近期股价走势
10. 风险与核查点
11. 来源说明
